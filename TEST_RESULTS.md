## 🔍 调试指南

### 如果模型未加载
1. 检查浏览器控制台错误
2. 验证 /models/Haru/ 路径
3. 确认文件权限正确
4. 检查 CORS 设置

### 如果动作不播放
1. 检查动作文件路径
2. 验证 model3.json 配置
3. 确认 Live2D 初始化顺序
4. 查看动画循环状态

### 如果表情不切换
1. 验证表达式文件路径
2. 检查 setExpression 方法调用
3. 确认表情索引正确
4. 测试单个表情加载
